= terrapin

* Yet another mass spec viewer.
* Uses Rubygame and FFI-OpenGL for simple scripting and FFI-Inliner for blazing fast speed.
* Meant to run any mzXML/mzML and mzIdentML.

== Example

    % terrapin

== Installation

Once I release it to rubygems

    gem install terrapin

Right now

    download/clone from github
    rake build
    gem install pkg/*.gem
